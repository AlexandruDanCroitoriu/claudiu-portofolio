Light, warm single-page Vue app for guitarist/composer Claudiu Marian Spataru. Built with Vite and Tailwind CSS.

## Active Sections
1) Hero: warm banded layout, large name/role, "View My Reel" CTA, featured photo.
2) About: left image, right soft card with concise bio.
3) Footer: socials + contact note.

## Tech
- Vue 3 (Composition API), Vite, Tailwind CSS 4.1.11, Vue Router 4.5.1
- HMR in dev

## Design Notes
- Palette: cream/amber gradients, soft tan borders `#F1DEC6`, slate text.
- Aesthetic: clean, modern, musical; generous whitespace; soft shadows.
- Components: HeroLight.vue, AboutLight.vue, FooterSection.vue.

## Data
- Central data in `src/data/claudiuData.js` (personal info, socials, copy strings).

## Deployment
- `vite.config.js` base set to `/claudiu-portofolio/` for GitHub Pages.
