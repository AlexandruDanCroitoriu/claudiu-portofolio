# Claudiu Marian Spataru — Guitarist Portfolio

Modern portfolio website for guitarist and composer Claudiu Marian Spataru. Built with Vue 3 and Vite for optimal performance and a sleek user experience.

## Overview

This single-page application follows a light, warm aesthetic inspired by the reference layout and showcases Claudiu's work, media, and contact in a clean, minimal presentation.

## Design Philosophy

- **Light, Warm Theme**: cream backgrounds with warm amber accents and slate text
- **Sexy & Modern**: Sleek typography, smooth animations, and contemporary layouts
- **Minimalistic**: Clean design with purposeful whitespace and focused content
- **Guitarist-Focused**: Content and imagery that resonates with guitar listeners, students, and collaborators

## Technology Stack

- **Framework**: Vue 3 with Composition API
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Architecture**: Single Page Application (SPA)

## Quick Start

```bash
npm install
npm run dev
```

## Website Sections

1. **Hero** – Large name, role label, “View My Reel” and featured photo
2. **About Me** – Left image with right soft card and short bio
3. **Footer** – Social links and additional information

## Target Audience

Guitar enthusiasts, practitioners, and individuals seeking professional guitar coaching or performances services.
